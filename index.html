How does java script work?


Ans: at first, the browser takes the html and css file from clients then parse it to machine readable language and keep it to the DOM(document object model). the DOM is a live presenter of html and css files which is displayed in the form of human understandable view through DOM. Then browser takes javascript files and execute the files synchronously. thats why it is called single threted language. this execution happened through interpreter engine. this engine vary from one browser to another. example: chrome uses v8 engine to run as a js interpreter. mojila use spider monkey. after execution js files via interpreter, it doesnt show in the DOM immediately. it waits for renderring the html and css files. afer completion of renderring, js is dynamically visible in the web page through DOM.
